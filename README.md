# 微信小程序之Lincoo's Blog #

## 关于作者 ##

* 我是一名PHP程序员，前端对我来说就是神学，抱着学习的态度做了这款小程序；
* 大体是将自己的Blog做成一个app，以便更好的查看、分享;
* 持续开发中，欢迎关注。

## 功能特点 ##

* 功能完备，实用导向
* Server端API支持
* 涵盖众多组件、API使用，适用于学习微信小程序
* 多行文本模拟实现
* tab切换
* 模态框
* 本地数据组织及存储
* 图片预览功能

## 使用步骤

1. 克隆代码:

```bash
$ cd path/to/your/workspace
$ git clone https://github.com/harveyqing/BearDiary.git
```

2. 打开`微信Web开放者工具`（注意：须`v0.10.101100`及以上版本）

3. 添加项目

    * AppID：选`无AppID`
    * 项目名称：任意填写
    * 项目目录：path/to/your/workspace
    * 点击 `添加项目`
